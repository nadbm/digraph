# Digraffe

A simple app being used to learn about Elixir, Phoenix and Ecto.

## Setup

```
% git clone git@github.com:emwalker/digraffe.git
% cd digraffe
% mix deps.get
% npm install
% mix do ecto.create, ecto.migrate
% mix phoenix.server
% open http://localhost:4000/contexts  # in another terminal
```
